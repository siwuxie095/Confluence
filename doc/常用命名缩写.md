# 常用命名缩写



| 命名                | 缩写                           |
| ------------------- | ------------------------------ |
| Temporary           | temp 或 tmp                    |
| Class               | clazz 或 clss 或 cls 或 caller |
| Super               | supper 或 zuper                |
| Object              | obj                            |
| Primitive           | prim                           |
| Comparator          | comp 或 cmp                    |
| Comparable          | comp 或 cmp                    |
| Collection          | coll                           |
| List                | list 或 lst                    |
| Set                 | set                            |
| Map                 | map                            |
| Entry               | entry                          |
| Set<Map.Entry<K,V>> | entrySet                       |
| EntrySet            | entrySet                       |
